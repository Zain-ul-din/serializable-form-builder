# Serializable Form Builder

An in-app drag & drop form builder with serialization support to render forms dynamically while being compatible with shadcn/ui forms and Zod validation.

## Features

- **Drag & Drop Builder** — Compose forms by dragging field types onto a visual canvas with intuitive reordering
- **Zod Schema Generation** — Automatically generates Zod validation schemas from your form configuration
- **Serialization** — Export form definitions as portable, serializable JSON
- **Render Form** — Render fully functional forms compatible with shadcn/ui forms & Zod validation.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Zain-ul-din/serializable-form-builder.git
cd serializable-form-builder

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.
